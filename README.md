# QuizMaster
Веб-приложение для проведения квизов в реальном времени.  
Выполнено в рамках учебного проекта.

## Стек технологий
- **Фронтенд:** React, HTML, CSS, JavaScript
- **Бэкенд:** Node.js, Express.js
- **WebSocket:** Socket.IO
- **База данных:** JSON-файлы

## Установка и запуск

### 1. Установите Node.js
Скачайте с [nodejs.org](https://nodejs.org) (версия LTS)

### 2. Скачайте проект
git clone https://github.com/TeshevaKristina/quizmaster.git
cd quiz_app

### 3. Установите зависимости сервера
npm install express socket.io cors

### 4. Запустите сервер
node server.js

### 5. В новом терминале — клиент
cd client
npm install
npm start

Приложение откроется на **http://localhost:3000**

## Как тестировать
1. Зарегистрируйте **организатора** (роль: Organizer)
2. В другой вкладке зарегистрируйте **участника** (роль: Participant)
3. Организатор создаёт квиз, добавляет вопросы и публикует
4. Организатор видит код комнаты — участник вводит его и подключается
5. Организатор нажимает «Начать квиз»
6. После ответов на все вопросы отображается лидерборд

## Функциональность
- Регистрация и авторизация (организатор / участник)
- Создание, редактирование, удаление и перезапуск квизов
- Вопросы: текстовые и с картинкой, одиночный и множественный выбор
- Комнаты ожидания с уникальным кодом
- Вопросы в реальном времени с таймером (WebSocket)
- Автопереход при ответе всех участников
- Система подсчёта баллов с учётом времени ответа
- Лидерборд с медалями и временем
- История проведённых игр с возможностью очистки
- Запуск квиза без участников (для предпросмотра)
